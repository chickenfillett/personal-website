# DeskHaven image assets

DeskHaven images are organized by website locale:

- `zh`
- `en`
- `ja`
- `ko`
- `fr`
- `de`
- `es`

Each locale folder contains:

- `posters/poster-01.webp` ... `poster-10.webp`
- `screenshots/screenshot-01.webp` ... screenshots for that locale
