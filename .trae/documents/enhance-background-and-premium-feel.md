# 提升网页高级感 — 背景光晕与质感增强计划

## 目标

为 Chicken Fillet 个人品牌主页添加克制而高级的视觉质感，在保持极简、暗色、不喧哗的设计哲学前提下，通过多层 subtle 效果提升整体品质感。

## 现状分析

当前网站是纯扁平暗色设计：
- 背景：纯色 `#0a0a0a`
- 卡片：纯色 `#1a1a1a`
- 无渐变、无阴影、无动画
- 视觉层次仅靠色块和留白区分

## 实施方案

### Phase 1: 背景光晕系统（核心）

在全局背景层添加 2-3 个大型、低透明度、缓慢漂移的彩色光晕，营造空间深度感。

- **光晕数量**：3 个
- **颜色**：极低饱和度的蓝灰/青灰色调（如 `#1a2332`、`#1c2a2a`、`#232332`），与现有暗色主题融合
- **尺寸**：直径 400-800px 的圆形渐变
- **透明度**：3-8%，极其克制
- **运动**：CSS 动画，缓慢漂移（20-40s 一个周期），无规则路径
- **模糊**：高斯模糊 80-120px，边缘柔和
- **位置**：固定在背景层，不随内容滚动（或极慢视差滚动）
- **实现方式**：纯 CSS `@keyframes` + `position: fixed` 的 div 元素，无需 Canvas/WebGL

### Phase 2: 微妙纹理叠加

在纯色背景上添加几乎不可见的噪点纹理，消除数字屏幕的"死黑"感。

- **纹理**：SVG 噪点滤镜或 CSS 生成的细微颗粒
- **透明度**：2-5%
- **混合模式**：`overlay` 或 `soft-light`
- **目的**：增加"胶片感"和物理质感，让暗色背景更"活"

### Phase 3: 卡片质感提升

现有卡片是纯平色块，添加极微妙的层次区分。

- **边框**：1px `rgba(255,255,255,0.03)` 的极淡边框
- **背景**：从纯色 `#1a1a1a` 改为极微妙的渐变（如 `#1a1a1a` 到 `#151515`）
- **hover 状态**：边框透明度提升到 0.08，背景亮度极微提升
- **禁止**：不使用阴影、不使用圆角变化、不使用位移动画

### Phase 4: 文字与排版细节

- **Hero 标题**：添加极微弱的文字渐变（从 `#f0f0f0` 到 `#c0c0c0`），增加立体感
- **大标题字间距**：微调 tracking，让标题更舒展
- **行高优化**：长文本行高从 1.5 提升到 1.65-1.75，阅读更舒适

### Phase 5: 微交互（克制版）

- **链接 hover**：颜色过渡保持 0.2s，但添加极微的 `translateY(-1px)`
- **按钮 hover**：背景色反转（现有），但添加 0.15s 的 ease-out 过渡
- **页面加载**：内容区域极淡的 fade-in（0.3s），非逐个元素动画
- **禁止**：不使用弹跳、不使用缩放、不使用旋转

## 技术实现要点

1. **性能优先**：所有效果使用 CSS 实现，不使用 JavaScript 动画库
2. **可关闭性**：光晕系统作为独立组件，方便未来调整或移除
3. **响应式**：光晕在移动端缩小尺寸，避免遮挡内容
4. **无障碍**：所有效果不影响内容可读性，对比度保持 WCAG 标准
5. **静态导出兼容**：效果不依赖服务端，支持 `output: 'export'`

## 验收标准

- [ ] 背景有 2-3 个缓慢漂移的柔和光晕
- [ ] 光晕颜色与暗色主题融合，不突兀
- [ ] 背景有极微妙的噪点纹理
- [ ] 卡片有极淡边框和微妙背景层次
- [ ] Hero 标题有微弱的文字渐变
- [ ] 所有 hover 交互更细腻
- [ ] 移动端效果适配正常
- [ ] 构建无错误
- [ ] 性能无退化（Lighthouse 性能分数保持 95+）
